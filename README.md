# HUNTER×HUNTER タイピングマスター 🎮⌨️

> 念能力を習得するように、タイピングを習得しよう！

HUNTER×HUNTERの世界観をモチーフにしたタイピング練習ゲームです。PWA対応でスマートフォンでもプレイ可能です。

## 🎯 特徴

- **HUNTER×HUNTER世界観**: キャラクター名、念能力、用語をタイピング
- **段階的な学習**: 念の修行のように基礎から応用へステップアップ
- **複数入力パターン対応**: 「し」→ si/shi/ci など柔軟な入力判定
- **PWA対応**: オフラインでもプレイ可能
- **レスポンシブ**: PC・タブレット・スマホ対応

## 🛠️ 技術スタック

- **フレームワーク**: React 18 + TypeScript
- **ビルドツール**: Vite
- **状態管理**: Zustand
- **スタイリング**: Tailwind CSS
- **アニメーション**: Framer Motion
- **音声**: Howler.js
- **PWA**: vite-plugin-pwa
- **テスト**: Vitest + Testing Library

## 🎮 ゲームモード

### トレーニングモード
- 6章構成のストーリーモード
- 各章6ステージ、段階的に難易度アップ
- ステージクリアで次のステージ解放

### タイムアタック
- 60秒間で何文字打てるか挑戦
- ランキング機能（予定）

### フリープレイ
- 好きなカテゴリの単語で練習
- 難易度調整可能

## 🔧 ローマ字エンジン

複数の入力パターンに対応した高精度なローマ字判定エンジンを搭載。

### 対応パターン例

| ひらがな | 入力パターン |
|---------|-------------|
| し | si, shi, ci |
| ふ | hu, fu |
| っか | kka, xtu+ka |
| ん | n (子音前), nn (母音前) |

### パフォーマンス目標
- 入力判定: 16ms以内
- テストカバレッジ: 95%以上
